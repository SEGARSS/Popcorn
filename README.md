<summary> План работы по проекту 


###

- [Конец уровня при потере мяча]()
    - Анимация расплавления платформы.
    - Анимация выкатывания новой платформы.
    - Анимация расширяющейся платформы.
    - Пуск мяча пробелом и состояние мяча.

###

- [Кирпичи]()
    - Обычные кирпичи(синие или красные).
        - Могут быть с буквой или без.
        - При попадании в кирпич он исчезает из уровня сразу, и если кирпич был:
        - Без буквы - создаётся активный кирпич
        - С буквой - создаётся падающая буква.
    - Неразрушаемый кирпич.
        - Анимация при попадании.
    - Многоразовый кирпич.
        - 4 состояния кирпича (1-4 удара до разрушения)
        - Переход по состояниям.
        - Анимация приза при разрушении (кирпич исчезает, анимация - остаёться).
    - Кирпич с парашютом.
        - Анимация парашюта - как вариант падающей буквы.
    - Кирпич телепортации.
        - Анимация при захвате и выске мяча. 
    - Кирпич рекламы.
        - Анимация фрагмента "рекламы", скрывающая за кирпичом.  

###

- [Взаимодействие особых кирпичей]()
    - Падающая буква - взаимодействует только с платформой.
    - Активный кирпич взаимодействует только с мячиком

###

- [Действия букв]()
    - Простые.
        - О ("Отмена") - отмена действия символ К,Ш,П,Л,М.
        - И ("Инверсия")
        - С ("Скорость")
        - М ("Монстры")
        - Ж ("жизнь")
    - Сложные.
        - К ("Клей")
            - Анимация растекающегося клея (прямая и обратная)
            - Фиксация шарика (+ новое состояние мячика)
            - Пуск мячика пробелом и по таймеру
        - Ш ("Шире")
            - Новые состояние платформы.
            - Анимация расширения платформы (пряма и обратная)
        - П ("Пол")
            - Отражение шарика от пола.
            - Шкала пола
            - Взаимодействия мячика с полом.
            - Отбивание мяча от нижней грани платформы
        - Л ("Лазер")
            - Новое состояние платформы
            - Анимация превращения в пушку (прямая и обратная)
            - Стрельба пробелом.
            - Обработка пробела особым образом, когда платформа в режиме лазера.
            - Объекты лазерных лучей - анимация, перемещение и взаимодействие с другими объектами.
        - Т ("Три")
            - Добавить поддержку множества мячиков.
            - Вывод множество мячиков.
            - Взаимодействие множества мячиков с элементами игры.
        - "+" ("Переход на следующий уровень")
            - Анимация досрочного перехода на следующий уровень.
            - Отдельный уровень перехода с особыми стенками (рамкой)

###

- [Движение Мячика]()
    - Отскок мячика от рамки.
    - Попадания в края платформы.
    - Подсечка мяча платформой.
    - Попадание в монстра.

###

- [Попадание в кирпич]()
    - Попадание в кирпич - состояние кирпича.
    - Выбивание падающей буквы.
    - Перехват падающей буквы.
    - Для каждой буквы есть свои действия.

###

- [Гейты]()
    - Вывод гейтов.
    - Анимация открытия/закрытия гейтов.
    - Состояния гейтов (закрыт, откорывается, закрывается).

###

- [Монстры]()
    - Список монстров 
        - "глаз" 
        - "черепаха" 
        - "сатурн" 
        - "голова" 
        - "бубен" 
        - "аквариум" 
        - "зажигалка"
    - Для каждого сделать раскодировку.
    - Перемещение, выбор направления + состояния(живой/пауза/исчезание).
    - Взаимодействие с мячом и платформой.
    - Выход из гейт.
    - Для каждого монстра анимация 

###

- [Информационная панель]()
    - Логотип.
    - Индикаторы.
        - Имя игрока.
        - Текущий счёт.
    - Шкала монстров и пола.
    - Изменение шкал.
    - Окно дополнительных жизней.
    - Учёт игровых действий и отображение на индикаторах.

###

- [Игра и уровни]()
    - Список первых 10 уровней.
    - Состояние игры.
    - Заставка анимация начала уровня.
    - Играем уровень.
    - Потеря жизни.
    - Переход на следующий уровень (нормальный и досрочный)
    - Окончание игры.
