* Источник обучающих материалов, какнал youtube Alexander Semenko  - [Ссылка](https://www.youtube.com/@AlexanderSemenko)

### План работы по проекту 

[Текст]()

    1. Анимация расплавления платформы.

    1. Отскок мячика от рамки.

    3. Подсечкеа мяча платформой.
    4. Попадание в монстра.
    
[Попадание в кирпич]()

    1. Попадание в кирпич - состояние кирпича.
    2. Выбивание падающей буквы.
    3. Перехват падающей буквы.
    4. Для каждой буквы есть свои действия.

    1. Список монстров.

    5. Выход из гейт.
